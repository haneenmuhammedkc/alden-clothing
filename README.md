# Alden Clothing - MERN E-commerce

Full-stack e-commerce web app with Admin Panel and Razorpay Integration.

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- Firebase
- Razorpay

## Features
- User Authentication
- Cart & Checkout
- Online Payments
- Admin Dashboard

## Folder Structure
frontend/ → React App  
backend/ → API Server  

## Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
